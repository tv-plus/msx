Установка Lampa через Media Station X

Установка и настройка Media Station X Для телевизоров Samsung и LG !!!

1. Зайдите в "магазин" вашего устройства и скачайте приложение "Media Station X"
2. Запустите приложение Media Station X
3. Откройте настройки (Settings) - (Start Parameter) - (Setup)
4. Введите в поле адрес плейлиста: Lampa с рекламой официальная lampa.mx
5. Введите в поле адрес плейлиста: Lampa без рекламы bylampa.line.pm
6. Далее если не можешь настроить то жмити кномпу Помощь в настройке !!!
